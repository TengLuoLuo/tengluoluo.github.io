博客搭建思路

​ 该博客主要思路是运用 Github Pages + Hexo 来实现的。

相关工具及技术介绍

  • Github

  • Git

  • Github Pages

    • Github Pages 提供免费的静态网页托管服务，可以通过 Github Pagses 托管博客、项目官网等静态网页。
    • 登录 Github 并创建新仓库，名称格式为 xxx.github.io，xxx 为账户名，那么这个仓库会被识别为 Github Page 服务。

  • Hexo

    • Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，并利用靓丽的主题生成静态网页。本博客便是借由Hexo搭建而成。

    • 博客搭建后其文件夹一般包括_config.yml（存储网站的配置信息）、package.json（应用程序的信息）、sacffolds（模板文件夹） 、source（资源文件夹存放用户资源） 、themes（主题文件夹） 等。

    • 常用命令：

      •  hexo new <标题> 新建文章
 hexo generate 生成静态文件
 hexo publish 发布草稿
 hexo server 启动服务器。默认情况下访问网址为http://localhost:4000/
 hexo deploy 部署网站
 hexo clean 清除缓存文件
      • More info: Writing、Server、Generating、Deployment
