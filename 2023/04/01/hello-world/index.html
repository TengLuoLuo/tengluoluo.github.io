技术报告

相关工具及技术介绍

  • Github

    • GitHub 是一个基于 Web 的代码托管平台，允许开发者将代码存储在云端并进行版本控制，这使得开发者可以协作开发、共享代码并控制代码的变更。GitHub 成立于 2008 年，现在已经成为全球最大的开源社区之一，拥有数千万的注册用户和数百万的开源项目。

  • Git

    • Git 是一个分布式版本控制系统，它允许开发者对代码进行版本控制，并跟踪代码的变化。Git最初用于管理 Linux 内核代码。现在，Git 已经成为广泛使用的版本控制系统，不仅仅用于 Linux 内核，还用于管理各种开源项目和私有代码库。

  • Github Pages

    • Github Pages 提供免费的静态网页托管服务，可以通过 Github Pagses 托管博客、项目官网等静态网页。
    • 登录 Github 并创建新仓库，名称格式为 xxx.github.io，xxx 为账户名，那么这个仓库会被识别为 Github Page 服务。

  • Hexo

    • Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，并利用靓丽的主题生成静态网页。本博客便是借由Hexo搭建而成。

    • 博客搭建后其文件夹一般包括_config.yml（存储网站的配置信息）、package.json（应用程序的信息）、sacffolds（模板文件夹） 、source（资源文件夹存放用户资源） 、themes（主题文件夹） 等。

    • 常用命令：

      • hexo new <标题> 新建文章
        hexo generate 生成静态文件
        hexo publish 发布草稿
        hexo server 启动服务器。默认情况下访问网址为http://localhost:4000/
        hexo deploy 部署网站
        hexo clean 清除缓存文件
      • More info: Writing、Server、Generating、Deployment

  • Typora

    • Typora 是一个免费的开源文本编辑器，支持多种编程语言和格式，并且可以在多个平台上使用。Typora 的设计目标是简单、高效和易于使用。它支持 HTML、CSS、JavaScript、TypeScript、Python、Perl、Ruby 等编程语言，并且可以自动检测并加载相关语法文件，使开发者可以更轻松地编写代码。Typora 还提供了许多其他功能，例如代码折叠、自动完成、代码片段、查找和替换等。

  • 选取标准

    ​ 博客主题选取标准有很多，我主要考虑的因素有以下几点：

    • 个人兴趣和爱好：博客作者应该选择自己感兴趣和擅长的主题，这样可以保证写作的动力和热情。
    • 写作能力和风格：选择适合自己写作能力和风格的主题，可以更好地发挥个人才华，同时也更容易吸引志同道合的读者。
    • 时间和能力：选择自己能够长期坚持写作的主题，可以保证博客的长期发展和稳定。

  • 个性化修改

    • 可以对选取主题下的config配置文件进行修改，修改一些必要的文字，如主页正中间的文字等。

作业总结&反思

  • 此次作业让我加深了对github、git、hexo等各项工具及技术的使用，下面是我的一些总结和反思：

  • 学习成本：使用 GitHub Pages 和 Hexo 需要一定的学习成本，特别是对于那些没有使用过 Git 和 HTML 编辑器的人来说。不过，一旦掌握了这些工具的基本使用方法，就可以快速创建和发布博客。

  • 自定义程度:GitHub Pages 和 Hexo 都提供了许多自定义选项，用户可以自定义网站的样式、主题和插件等。但是，GitHub Pages 的自定义程度更高，可以很好地与 Git 工具集成，而 Hexo 的自定义程度更高，需要用户自己编写 HTML 和 CSS 代码。

  • 响应式设计:GitHub Pages 和 Hexo 都支持响应式设计，用户可以根据需要设置网站的响应式布局。但是，GitHub Pages 的响应式设计功能更强大，可以自动适应不同的设备尺寸。而 Hexo 则需要使用特定的插件来实现响应式设计。

  • 安全性：使用 GitHub Pages 和 Hexo 搭建博客都需要用户注意网站的安全性，特别是对于那些使用免费域名的人来说。需要定期更新和备份网站数据，并使用安全的密码和访问权限等。

  • 博客内容：无论使用哪种工具搭建博客，都需要重视博客内容的质量和价值。只有为用户提供有用的信息和资源，才能获得用户的认可和支持。同时，还需要注重博客内容的更新和发布频率，以保持博客的活力和影响力。

总的来说，使用 GitHub Pages 和 Hexo 搭建博客可以帮助用户快速创建和发布自己的博客，但是需要一定的学习成本和自定义程度。用户可以根据自己的需求和偏好选择适合自己的工具，并注重博客内容的质量和价值，以获得更多的支持和认可。
